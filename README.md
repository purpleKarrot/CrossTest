CrossTest
=========

Minimal example project that:

*   Uses [CMake](http://cmake.org/)
*   Uses [Boost](http://boost.org/) (header only)
*   Runs tests with [Wine](http://www.winehq.org/) when crosscompiled on Linux for Windows
